# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by opening 
a GitHub Issue at:
https://github.com/OpenHouze/OpenHouze.github.io/issues

Please include the following information in your report:
- Description of the vulnerability
- Steps to reproduce the issue
- Potential impact of the vulnerability
- Any suggested fixes (if applicable)

## Response Time
We will acknowledge your report within 48 hours and work to address 
any confirmed vulnerabilities as quickly as possible.
